欢迎来到我的个人博客！

这是一个使用 [Hugo](https://gohugo.io/) 静态站点生成器和 [Ladder 主题](https://github.com/guangzhengli/hugo-theme-ladder) 构建的博客网站。

## 关于这个博客

在这里，我会分享：

- 💻 **技术文章**：编程、开发工具、最佳实践
- 📚 **学习笔记**：读书心得、课程总结
- 🎨 **生活随笔**：日常思考、旅行见闻
- 🔧 **项目经验**：开源项目、个人作品

## 开始使用

如果你也想搭建类似的博客，可以：

1. Fork 这个仓库
2. 修改 `config.yml` 中的个人信息
3. 在 `content/blog/` 目录下创建你的文章
4. 通过 GitHub Actions 自动部署到 GitHub Pages

## 特色功能

- ✨ 支持暗黑模式和明亮模式切换
- 🌍 多语言支持（中英文）
- 📱 响应式设计，移动端友好
- 🎯 专注于阅读体验的简洁设计
- 🏷️ 标签和归档功能
- 💬 留言板功能

感谢访问，希望你能在这里找到有价值的内容！
